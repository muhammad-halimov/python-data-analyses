# Installing & Running
- Install Python
- Create virtual environment (venv):
```
python3 - Linux/macOS
python - Windows NT
pip3 - Linux/macOS
pip - Windows NT
```
```
python -m venv venv
```
- Activate virtual environment (venv):
```
venv/Scripts/activate - Windows 
```
```
source venv/bin/activate - GNU/Linux & macOS/BSD
```
- Followed by instuctions:
```
pip install -r requirements.txt
```
```
python main.py
```
